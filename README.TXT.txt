Este programa administra eventos que pueden ser guardados, editados, leidos y eliminados para una agenda personal.

Se puede añadir un nuevo evento a un determinado mes, al hacer click en el mes correspondiente.

El programa está diseñado para que respete la cantidad de días que cada mes tiene y avise al usuario en caso de estar equivocado
a la hora de usarlo.


Cada evento tiene una fecha y una breve descripción.


las secciones para borrar y editar un evento son accesibles por medio de botones justo en la fila de cada evento para su facil
acceso.

La base de datos utilizada está en el archivo sql que acompaña este proyecto, el lenguaje utilizado mayoritariamente fue PHP
y se utilizo bootstrap para partes del diseño responsive de la aplicación.
